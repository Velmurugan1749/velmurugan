HTML

HTML stands for HyperText Markup Language.It is sued to design the web pages using a markup language.HTML is a combination of Hypertext and Markup
Language.Hypertext defines the link between web pages.A markup language is used to define the text document within the tag which defines the
structure of web pages.This language is used to annotate(makes note for the computer) text so that a machine manipulation has to be done on the text.
Most markup language are human-readble.The language uses tags has to be done on the text.

HTML is a markup languageused to manipulate text,images and other content,in order to display it in the required format.HTML was
created by Tim Berners Lee in1991.The first ever version of HTML was HTML 1.0,but the standard version was HTML 2.0 published in 1995.
